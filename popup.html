Verifica url do site tibia.com!

Informações


A verificação funciona de forma muito simples, ao entrar em um site que tenha a palavra "tipia" na url ele verifica se o site é "www.tibia.com", caso não seja ele mostra uma mensagem de aviso no topo da pagina.

Esse plugin esta sobre a licensa MIT. Para contribuir com esse projeto acesse o repositório do GitHub.
